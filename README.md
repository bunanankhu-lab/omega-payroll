# โอเมก้าอีสาน — ระบบจัดการเงินเดือนพนักงาน (Payroll)

เว็บแอพไฟล์เดียว ทำงานในเครื่อง (ไม่ต้องมีเซิร์ฟเวอร์) เก็บข้อมูลด้วย `localStorage` ของเบราว์เซอร์

## ไฟล์ในโปรเจกต์
- **payroll.html** — ระบบเงินเดือน: ทะเบียนพนักงาน, ทำเงินเดือนรายเดือน (OT / เบี้ยเลี้ยง / ประกันสังคม / หักลา-ขาด-สาย),
  นำเข้าไฟล์สแกนนิ้ว (.xls) เพื่อคำนวณ OT และมาสายอัตโนมัติ, พิมพ์สลิปเงินเดือน, ธีมสีบริษัท + โหมดมืด/สว่าง
- **รายรับรายจ่าย.html**, **income-expense.html** — แอพบันทึกรายรับ-รายจ่ายส่วนตัว (เวอร์ชันทดลอง)
- **xlsx.full.min.js** — ไลบรารี [SheetJS](https://sheetjs.com) สำหรับอ่านไฟล์ Excel (.xls/.xlsx) ในเบราว์เซอร์
- **logo.png** — โลโก้บริษัท

## วิธีใช้
เปิด `payroll.html` ด้วยเบราว์เซอร์ (ดับเบิลคลิก) — ต้องมี `xlsx.full.min.js` และ `logo.png` อยู่โฟลเดอร์เดียวกัน

### นำเข้าไฟล์สแกนนิ้ว
1. แท็บ **นำเข้าสแกนนิ้ว** → เลือกไฟล์รายงาน (.xls) จากเครื่องสแกน
2. ตรวจตารางพรีวิว → กดยืนยัน (จับคู่ด้วยรหัสพนักงาน, สร้างพนักงานใหม่อัตโนมัติ)
3. แท็บ **พนักงาน** เติมเงินเดือน → แท็บ **ทำเงินเดือน** กด "คำนวณ OT/สาย จากสแกนใหม่" → บันทึก
4. แท็บ **สลิป** เลือกพนักงาน → พิมพ์

## เกณฑ์การคำนวณ (ปรับได้ในหน้าตั้งค่า)
- เวลาทำงาน **08:30–17:30**
- **OT** = ชั่วโมงหลังเลิกงาน × ค่าจ้าง/ชม. × **1.5** (ค่าจ้าง/ชม. = เงินเดือน ÷ 30 ÷ 8 · ปัดลงเป็นชั่วโมงเต็ม)
- **ประกันสังคม** 5% สูงสุด 750 บาท/เดือน
- **หักมาสาย** ขั้นบันได: 1–150 นาที = 3฿/นาที · 151–300 นาที = 5฿/นาที · เกิน 300 นาที = 8฿/นาที

## หมายเหตุความเป็นส่วนตัว
ข้อมูลพนักงานและเงินเดือนถูกเก็บใน `localStorage` ของเบราว์เซอร์เครื่องที่ใช้งานเท่านั้น
**ไม่ได้อยู่ในซอร์สโค้ดและไม่ถูกอัปขึ้น GitHub** — ไฟล์รายงานสแกนนิ้ว (`*.xls/*.xlsx`) ถูกกันไว้ใน `.gitignore` แล้ว
